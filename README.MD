A fun and interactive **Python Quiz Game** where players answer multiple-choice questions under a **10-second time limit**. Scores are tracked and displayed on a **leaderboard**, ensuring only the highest score per player is saved.

---

## ✨ Features
- 🎲 Randomized question order for each playthrough  
- ⏰ 10-second timer per question  
- ✅ Instant feedback on correct/incorrect answers  
- 🏆 Leaderboard system (Top 5 scores displayed)  
- 📂 Persistent score storage in `leaderboard.txt`  
- 🔄 Updates only when a player achieves a higher score
